## 풀이한 문제 ⚒️
- 입력 받은 N개의 수 중에서 소수인 수의 개수를 찾아 출력하는 문제
- 알고리즘 분류: 수학, 정수론, 소수 판정

## 소요 시간 🛩️
- [X] 1시간 미만
- [ ] 1~3시간
- [ ] 3시간 이상

## 문제풀이 중점사항 🤔
```
// 소수 판별 함수
bool isPrime(int n) {
    if (n <= 1) return false; // 1 이하 는 소수가 아님

    // 2부터 n의 제곱근까지 나누어 떨어지는지 확인
    // 그 이상의 수는 중복되므로 확인할 필요 없음
    for (int i = 2; i <= sqrt(n); i++) {
        if (n % i == 0) {
            return false;
        }
    }
    return true;
}
```
